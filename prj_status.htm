Change Log

  • Release 1.0 - March 31, 2011
    • Corrected issue with Microsoft.Jet.OLEDB.4.0 driver that affected null TEAM column values in XML app.
    • Added keyboard event controls to Flash presention - can use keyboard or mouse
  • Beta Patch - March 17, 2011
    • Contains only the XML application.
    • Modified query to allow display of contestants that do not have a cooresponding category in Sheet1 of the spreadsheet.
    • Add prompt to alert user that not all elements of the data file were loaded
    • Added highlighting to the cell elements where an image file name was not found for the category providing a visual to the user.
    • Most of the missing category/images are naming variations that can be easily corrected by adding/editing the additional category in Sheet1 of the spreadsheet.
    • Some examples include:
      • Quiz Bowl - Alternate - There was not a Quiz Bowl - ALTERNATE category. Correct by adding the additional category variation to Sheet1 of the spreadsheet and use the same image as the Quiz Bowl
      • HVAC - The previous category was H V A C with spaces. Edit either the categories list(Sheet1) or the the contestants list (Sheet2) to use the same spelling
      • Mansonry - HELPER - The previous category was Masonry - helper.
    • O H & S categories do not have an category image and I did not notice one in the previous powerpoint that I could use. Correct this by:
      • adding each O H & S variation to Sheet1 of the spreadsheet
      • add the image file name(s) that you wish to use
      • Add the image to the _images directory (170w x 170h recommended)
  • Beta Version - March 11, 2011
    • End user help file created.
    • Corrected XML app datagrid resize to window form issue.
    • Added presentation run event to XML app to allow presentation to be started from one place. (requires flash projector exe)
    • Multi-Level Awards and Prize graphics completed for presentation.
    • Presentation animations completed.
    • Added Team text display to presentation
    • Added dynamic control of student text field font size for multi-student display. Font size reduces as the number of students on the team are increased.
  • Alpha Version - March 1, 2011
    • Working XML generator application compiled for .NET 3.5 framework
    • Rough draft of Flash presentation
    • Sample spreadsheet for use as data source.
