Starting the Presentation

SkillsUSA Presentation Project


The presentation can be started either from the XML application or by double clicking the SkillsUSA_Presentation.exe from the file system.

When the presentation is first started, it will automatically read the data file and generate a start up slide displaying the SkillsUSA logo.


The logo is displayed indefinitely to allow the presenter time for any last minute preparations that are needed.


When you are ready to start, click any area of blank area of the presentation.


Each slide will initially display the Category, the Category image, and the Award and/or Prize status in a series of animations. During this animation, the navigational controls are disabled and will not allow the next slide to be loaded until the Category sequence has completed.


After the Category sequence completes, the presentation pauses to allow the presenter the opportunity to make any announcements.


To display the School and Student, click any blank area of the presentation or press the UP key on the keyboard.


After the School and Student animations have been display, the presentation will pause again providing the presenter control over the flow of the presentation.


To continue the presentation, click one of the navigation icons located toward the bottom right corner of the screen or use the LEFT or RIGHT keyboard keys.


The navigation control also provides a slide status indicating the current slide being displayed and the total number of slides in the presentation.
