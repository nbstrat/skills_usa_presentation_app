Set Up

SkillsUSA Presentation Project


     Create a new directory on your system such as SkillsUSA_Project.

     Unzip the package into the directory that you created.


Your directory structure should now resemble the following:

The directory you created...

       

       

       skills_usa_data.xml


This is the data file that the SkillsUSA.exe application creates.


       SkillsUSA.exe


This is the application that reads the XLS spreadsheet and allows you to assign Awards and Prizes to the various students for use in the presentation.


       SkillsUSA.exe.config

               

Configuration file for SkillsUSA.exe.


       SkillsUSA_Presentation.exe

               

This is the presentation application that uses the xml file created with the SkillsUSA application.

       

       _docs

               

This help file.

       _images


Contains all the Category images used in the presentation. A new        Category can be added to the presentation by adding the entry into the spreadsheet  and placing the Category image here.

       _skills_usa_test_data

               

Sample spreadsheets used during development. You should use one of these as your template in preparing your data. (test_case_teams.xls)

       _source

       

The _source directory contains the source code created to make this project. These directories have sub-directories, but they are not important unless you want to start additional development. You can safely delete the _source directory if you wish.
